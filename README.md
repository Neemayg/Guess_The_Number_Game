# Appirates_Web_Dev

## Day 2 Project: Number Guessing Game

A simple browser-based number guessing game built with HTML, CSS, and JavaScript.

### Features

- Random number generation (1-20)
- Score tracking
- High score tracking
- Real-time feedback (Too high, Too low, Correct)
- Game reset functionality

### How to Play

1. Enter a number between 1 and 20
2. Click "Check!" to submit your guess
3. Follow the feedback (Too high/Too low)
4. Try to reach the correct number before running out of attempts
5. Your high score is saved during the session

### Files

- `index.html` - Game structure
- `script.js` - Game logic
- `style.css` - Game styling

### To Run

Open `index.html` in your web browser.
